# Static Images Directory

## Required Images

Place the following images in their respective directories:

### Core Images
- `logo.png` - EcoLearn platform logo (32x32px recommended)
- `favicon.png` - Browser favicon (16x16px or 32x32px)
- `hero-illustration.svg` - Hero section illustration

### User Avatars
- `avatars/default.png` - Default user avatar (128x128px)

### Badge Icons
- `badges/common.png` - Common badge icon
- `badges/uncommon.png` - Uncommon badge icon  
- `badges/rare.png` - Rare badge icon
- `badges/epic.png` - Epic badge icon
- `badges/legendary.png` - Legendary badge icon

### Topic Icons
- `icons/climate-change.png`
- `icons/renewable-energy.png`
- `icons/waste-management.png`
- `icons/biodiversity.png`
- `icons/water-conservation.png`

## Image Guidelines
- Use PNG format for icons and logos
- Use SVG format for illustrations when possible
- Optimize images for web (compress file sizes)
- Use consistent styling across all icons
- Ensure accessibility with proper alt text
