# Templates Directory

This directory contains all Django HTML templates organized by app.

## Structure
- `base.html` - Main base template with navigation and footer
- `components/` - Reusable template components
- `core/` - Core app templates (home, dashboard, profile)
- `accounts/` - Authentication templates (login, register)
- `education/` - Educational content templates
- `gamification/` - Gamification feature templates
- `assessments/` - Quiz and assessment templates

## Template Features
- Bootstrap 5 integration
- Responsive design
- Accessibility features
- SEO optimization
- Modern UI components
